# Code of Conduct

This project adopts the **[Contributor Covenant, version 2.1][cc]**
as its code of conduct, in full and without modification.

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

Reading the full text at the link above is part of contributing here.
Per axiom C8, this project does not maintain a custom version — the
upstream document is the source of truth.

## Reporting

To report a concern, choose either channel:

- **Email**: conduct@round.online — read by project maintainers, kept
  confidential. Response within 5 business days.
- **GitHub**: open an issue with the label `coc-report`. Use this only
  for public-record cases; for anything private, email instead.

Reports are reviewed by the project maintainers. We follow the
"Enforcement Guidelines" section of the Covenant when deciding on a
response.

## Scope

This Code of Conduct applies inside this repository's GitHub spaces
(issues, pull requests, discussions, wiki) and any official project
channel where contributors interact as representatives of the project.

It does not regulate behaviour in unrelated spaces, but a pattern of
conduct elsewhere may be considered when evaluating a reported incident.
