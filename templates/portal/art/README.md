# art/

Drop image files here. Auto-discovered at runtime — no manifest, no
`include_bytes!`, no path constants.

This portal template doesn't load any image assets by default; the
walls and player are coloured cubes. Add your own PNG/WebP for
sprites, textures, or atlases as your game grows.

See [the blank template's art/README.md](../../blank/art/README.md)
for the format and naming conventions every trikala project follows.
